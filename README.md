## Auth Demo

React + TypeScript + Vite

### Getting Started

---

- Clone the repository

```sh
git clone https://github.com/PaulAroo/authDemo.git
```

- Install dependencies

```sh
pnpm install
```

- Run the development server:

```sh
pnpm dev
# or
npm run dev
# or
yarn dev
# or
bun dev
```

Open http://localhost:5173 with your browser to see the result.
